# pastebin

A small offline pastebin app built with HTML, CSS, and vanilla JavaScript.

## Features

- Paste text, HTML, URLs, images, and files.
- Save items locally in IndexedDB.
- Search, sort, favorite, pin, rename, copy, download, and delete items.
- Import and export saved data as JSON.
- Light and dark themes.

## Run

```powershell
npm run dev
```

Open:

```text
http://127.0.0.1:4173/
```

## Notes

- No backend or external services are used.
- Clipboard permissions depend on the browser.
- Data stays in the current browser profile.
